# simpleBlog
A simple blog based on node.js

Author: Shen Dong

# Intro
Simple blog used the following techniques:
   web framework: node.js/express.js/async.js
   authentication: passport.js
   db: mongodb/mongoose
   cache: redis
   front end: jQuery/bootstrap
   view engine: jade
